# Store Rating System - Full Stack Application

A comprehensive web application for managing and rating stores, built with Express.js, React, MySQL, and plain CSS.

## Features

### System Administrator
- View dashboard with statistics (total users, stores, ratings)
- Add and manage stores
- Add and manage users (normal users and admins)
- View all stores with ratings
- View all users with filtering and sorting
- View detailed user information

### Normal User
- Register and login
- View all registered stores
- Search stores by name and address
- Rate stores (1-5 stars)
- Update existing ratings
- View overall store ratings and personal ratings

### Store Owner
- Login to dedicated dashboard
- View average store rating
- See list of customers who rated their store
- View individual ratings from customers

## Tech Stack

**Backend:**
- Node.js with Express.js
- MySQL database
- JWT authentication
- bcrypt for password hashing
- express-validator for input validation

**Frontend:**
- React 18
- React Router v6
- Axios for API calls
- Plain CSS
- Context API for state management
